# Automatización Web - Registro PrestaShop con Cucumber

## Descripción
Prueba automatizada para el registro de usuarios en PrestaShop usando Selenium, Java y Cucumber.

## Tecnologías
- Java 17
- Selenium WebDriver
- Cucumber
- Maven
- JUnit

## Cómo ejecutar
1. Clonar el repo
2. Abrir en IntelliJ
3. Ejecutar la clase TestRunner.java
4. Ver el reporte en target/cucumber/cucumber-report.html

## Evidencia
Reporte con 100% de escenarios exitosos adjunto en la entrega.
